## 1. Project Overview
Google Drive, DropBox, NAS 와 같이 사용자 계정에 따라 분리하여 파일을 원격 저장하고 파일의 종류(컨텐츠) 에 따라서 로드 발란싱하여 맞는 스토리지에 저장한다.

**서비스 종류:**
- 로그인
- 계정 만들기
- 파일 업로드
- 파일 다운로드
- 파일 삭제
- 파일리스트 조회
